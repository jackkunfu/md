# 编译原理
```
每种计算机只懂自己独特的指令系统，只能直接执行用机器语言编写的程序
机器语言对计算机依赖性强、直观性差、工作量大、程序结构欠清晰

高级语言   -----编译程序--->   机器语言/汇编语言

高级语言程序执行阶段
  编译到机器语言： 编译阶段、运行阶段
  编译到汇编语言： 编译阶段、汇编阶段、运行阶段

过程
  词法分析：根据英语的词法规则，将字符串识别为一个一个单词
    源代码字符串从左到右扫描和分解，识别出一个个具有独立意义的单词符号
  语法分析：根据英语的语法规则，对词法分析后的字符串进行分析、识别，并作出正确的语法检查
    根据语法规则从单词符号串中识别出各种语法单位（表达式、说明、语句等），并进行语法检查，确保语法结构上的正确性
  语义分析和中间代码生成：
  代码优化：
  目标代码生成：
```
